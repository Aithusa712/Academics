Retrieve sPCC data for a specific month.

This function retrieves the sPCC data from the provided data map for the specified month and fills the output maps with speed, temperature, and solar data.

Parameters
dataThe sensor log data map to retrieve sPCC data from.
monthThe month of the data to retrieve (0-11 for January-December).
speedDataInOutput map to store speed data for the specified month.
tempDataInOutput map to store temperature data for the specified month.
solarDataInOutput map to store solar data for the specified month.
